## Manifest file 

Manifest is a YAML file describing a malware.

### Required tags

* `LastChangeTime`
* `SHA256`
* `TLP`
* `Filename(submission)`

### Optionnal tags

* `SHA512`
* `SHA384`
* `SHA1`
* `MD5`
* `FirstBytes`
* `AssociatedGroups`
* `Techniques`
* `Names`
* `Path(submission)`
